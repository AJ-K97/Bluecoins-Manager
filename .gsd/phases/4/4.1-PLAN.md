# Plan 4.1: Single Transaction Entry (Core + CLI)

## Objective
Implement the core logic to add a single transaction manually and expose it via the CLI. This serves as the foundation for the Telegram Bot's "Add Transaction" feature.

## Context
- `src/commands.py`: Needs `add_transaction` function.
- `main.py`: Needs `add` subcommand.
- `src/policy.py`: Use existing policy validation for manual entries.
