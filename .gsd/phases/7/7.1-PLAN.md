# Plan 7.1: Telegram CLI Parity & Interactive Review

## Objective
Enable full management of Accounts and Categories via Telegram, matching CLI capabilities. Implement the "Interactive Review Queue" to allow approving or editing transactions directly from chat using inline buttons and numbered selections.

## Context
- `src/bot.py`: Main entry point for Telegram logic.
- `src/commands.py`: Underlying logic for accounts/categories.
- `src/interactive.py`: Review queue logic (needs adaptation for Bot).
