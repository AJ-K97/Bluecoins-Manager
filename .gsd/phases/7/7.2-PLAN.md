# Plan 7.2: Conversational Intelligence (RAG & Coaching)

## Objective
Enable natural language queries ("How much spent...") and interactive AI coaching ("Why did you categorize X as Food?"). Link the Telegram Bot to the RAG pipeline powered by `LocalLLMPipeline`.

## Context
- `src/bot.py`: Main Telegram interface.
- `src/local_llm.py`: RAG logic (Retrieval + Answer).
- `src/ai.py`: Coaching logic (`discuss_transaction`).

<task type="auto">
  <name>Interactive Coaching Loop</name>
  <files>src/bot.py, src/ai.py</files>
  <action>
    - **Trigger**: When user changes a category during review (e.g., from "Food" to "Gifts").
    - **Bot asks**: "Why did you change this? I categorized 'Cafe X' as Food because [Reasoning]."
    - **User replies**: "This was a gift card purchase."
    - **Persistence**: Store reasoning in `AIMemory.reflection` and generate `LLMSkill` rule automatically.
    - **Acknowledge**: "Understood. 'Cafe X' + Gift Intent -> 'Gifts'. Saved."
  </action>
  <verify>
    - Correct a transaction via bot.
    - Provide coaching reasoning.
    - Verify `AIMemory` table has new reflection.
  </verify>
  <done>
    - Correction loop improves future accuracy.
  </done>
</task>
